# 大成ジュニアバレーボールクラブ クリスマス会 得点王アプリ

バレーボールクラブのレクリエーションで使用する得点管理アプリです。

## 使い方
1. **順位表**: 全種目の合計得点と現在のランキングをリアルタイムで確認できます。
2. **得点入力**: 各ゲームごとに、順位や成否を選択するだけで簡単にポイントが加算されます。
3. **データ保存**: 入力したデータはブラウザ（localStorage）に自動保存されるため、ページを閉じても安心です。

## ゲーム内容
- 🗣️ 伝言ゲーム
- 👁️ 動体視力ゲーム
- 🎭 ジェスチャーゲーム
- 🏃 リレー

## 技術スタック
- React
- Tailwind CSS
- Recharts (グラフ表示)
- Lucide / FontAwesome (アイコン)
